# Thinkn — Project Context

## Company
Thinkn is a fast-moving B2B technology company that builds and licenses
digital platforms for governments, enterprises, and startups.
Primary markets: Africa, Asia, Middle East.
Positioning: like Oracle, Accenture, EY — but faster, younger, more
affordable, no corporate jargon.

## Brand
- Personality: modern, slick, futuristic, dynamic, challenger energy
- Voice: clear and intelligent, no buzzwords, bold and confident
- Visual: geometric, minimalist, strong identity
- Inspiration: Apple, Uber, Spotify
- Colors: deep navy (#0A0F2E), orange accent, electric blue (#2D3FFF),
  red highlight (#FF2D20)
- Typography: bold geometric sans-serif, heavy headlines

## Design System Status
No formal DS yet. Brand foundations exist in Figma.
Goal: token-based design system from existing brand assets.
Token naming convention: Brand/Orange/500, Neutral/900, etc.

## Work Streams
- Website
- Pitch decks & presentations
- Marketing materials
- Design system (tokens first, then components)

## Rules
- All code, copy, and deliverables in English
- Use Figma MCP to read design context before implementing
- Token-first approach for all UI work
- Component naming follows Figma layer structure
- Commit messages in English, clear and descriptive
